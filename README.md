# 麻醉登记管理系统

《麻醉登记管理系统》是专门为医院麻醉科管理而开发的一款软件，用于记录麻醉信息、麻醉药信息等。

主要功能包括：

一、基本信息：麻醉方式、科别、非预期相关事件、员工信息

二、登记管理：麻醉信息登记

三、查询：按姓名查询、按住院号

四、统计：麻醉方式统计、麻醉ASA分级统计、术后镇痛统计、复苏效果统计、进入PAVU统计



# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
